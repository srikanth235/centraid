# Chrome Web Store listing

## Name

Centraid Companion

## Short description

Pair your browser with Centraid for explicit password filling and private web capture.

## Detailed description

Centraid Companion connects this browser to your own Centraid gateway, even when it is on another network. Pair once with a short-lived QR code, then choose a matching Locker login when you want it filled. Passwords and one-time codes appear only for that gesture; the extension does not keep an offline secret cache.

Capture the current page or selection into Tasks and Notes, or save a visible screenshot to Docs. Agenda quick-add and manual People capture follow the same app grants you already control in Centraid. Revoke the browser at any time from Settings → Devices; its next request fails and its local pairing state is purged.

## Single purpose

Provide an owner-controlled companion surface for explicit Locker fill and capture into the owner's Centraid apps.
